# A simple example of a preloaded keycloak and a flask app

- `docker` is required
- `keycloak` instance will be preloaded with 2 users and 2 groups
  - `user:user` in group `users`
  - `admin:admin` in group `administrators`

## Getting started

In two terminals :

- `just run-keycloak`
- `just run-flask`
